# SmartERP

## Project Overview

A modern cloud-based ERP platform inspired by Tally.

## Modules

- Authentication
- Dashboard
- Company Management
- Ledger Management
- Group Management
- Stock Group Management
- Stock Item Management
- Unit Management
- Customer Management
- Supplier Management
- Receipt Voucher
- Journal Voucher
- Sales Voucher
- Purchase Voucher
- Credit Note
- Debit Note
- Billing System
- Inventory Management
- Reports
- Keyboard Navigation System

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- Supabase
